\# Security Policy 🛡️



\## Supported Versions



Currently, only the latest release of \*\*Smart Screen Unlocker Recovery\*\* is supported with security updates and bug fixes.



| Version | Supported          |

| ------- | ------------------ |

| v1.0.x  | :white\_check\_mark: |

| Beta    | :x:                |



\## Reporting a Vulnerability



If you discover a security vulnerability, execution bug, or a potential exploit path within this toolkit, please \*\*DO NOT\*\* open a public issue. 



Instead, please privately report it directly to the lead developer:

\* \*\*Contact:\*\* Reach out via the official website contact form at \[alisakkaf.com](https://alisakkaf.com) or send a direct message on \[Facebook (AliSakkaf.Dev)](https://www.facebook.com/AliSakkaf.Dev).



We take security seriously and will attempt to acknowledge, review, and resolve any legitimate forensic or execution vulnerabilities as quickly as possible.
